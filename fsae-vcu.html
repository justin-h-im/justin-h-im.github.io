Project Description

Overview

The Vehicle Control Unit (VCU) is the center of the vehicle's electrical system. It is responsible for controlling the motor according to driver inputs while also managing safety systems such as ready to move and plausibility checks. The project utilizes ADC via GPIO to capture driver input and CAN bus to send torque requests to the inverter and the motor. The VCU is seamlessly integrated with a Cascadia PM100DX inverter and a Emrax 228 motor.

Key Features

  • CAN bus communication for system integration
  • Configurable motor parameters for performance tuning
  • ADC via GPIO to capture driver input
  • APPS and BSE plausiblity checks
  • Ready to Move System Implementation
  • Implementation of slew rate filters to ensure intentional motor movement
  • Integrated with Prototype VCU hat to consolidate wiring
  • Designed in both a Simple While Loop and an FSM architecture
