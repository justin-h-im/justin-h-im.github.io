Project Description

Overview

The FSAE Dashboard is a comprehensive telemetry system designed for real-time vehicle monitoring and diagnostics in Formula SAE racing. This project addresses the critical need for drivers and engineers to monitor vehicle performance during testing and competition.

Key Features

  • Real-time sensor data acquisition and processing
  • Custom GUI with touchscreen interface optimized for racing conditions
  • Comprehensive data logging and analysis capabilities
  • Wireless data transmission to pit crew for real-time monitoring
  • Battery voltage and temperature monitoring with alerts
  • Engine RPM and vehicle speed tracking
  • G-force and acceleration measurements

Technical Implementation

The system is built around an STM32F4 microcontroller running FreeRTOS for reliable real-time operation. It integrates multiple automotive sensors through CAN bus communication, providing a robust and scalable architecture. The touchscreen interface is designed for easy operation while wearing racing gloves.

Challenges & Solutions

One of the main challenges was ensuring reliable operation in the harsh racing environment with high vibration and temperature variations. This was solved through careful component selection, robust PCB design, and comprehensive testing under simulated racing conditions.

Results

The dashboard successfully provided real-time data during FSAE competitions, helping the team optimize vehicle performance and identify potential issues before they became critical. The system's reliability and ease of use contributed to improved lap times and vehicle consistency.
